# Maxila Dental
A professional landing page for dental healthcare services.

## Project Overview
This repository contains the source code for the Maxila Dental web presence. The project focuses on providing a clean, informative, and accessible interface for dental patients.

## Tech Stack
* **Frontend**: HTML5, CSS3
* **Typography**: Montserrat (Clean & Modern)
* **Interactivity**: Vanilla JavaScript
* **Assets**: Optimized imagery for web performance

## Key Features
* **Service Showcase**: Detailed information about dental treatments.
* **Responsive Layout**: Optimized for both mobile and desktop users.
* **Modern UI**: Utilizing geometric typography for a professional clinical feel.

## Project Structure
* `index.html` - The main entry point of the website.
* `style.css` - Custom styling and layout definitions.
* `script.js` - Basic interactivity and logic.
* `images/` - Directory for visual assets and brand photography.

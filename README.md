# Maxila Dental
A professional landing page for modern dental care services, designed with a focus on clean aesthetics and user experience.

## Brand Identity & Aesthetic
This project follows a high-end visual storytelling approach, utilizing:
* **Typography**: Montserrat (Clean, geometric, and professional)
* **Visual Style**: High contrast with earthy tones (Moody & Artistic)
* **Design Philosophy**: Minimalist layout to highlight clinical excellence and patient comfort.

## Tech Stack
* **Frontend**: HTML5, CSS3 (Custom Grid & Flexbox)
* **Interactivity**: Vanilla JavaScript
* **Deployment**: GitHub Pages

## Key Features
* **Visual Archivalist Gallery**: High-quality documentation of dental procedures and results.
* **Service Overview**: Detailed sections for pre-wedding dental treatments and general care.
* **Responsive Design**: Fully optimized for mobile and desktop viewing.
* **Direct Booking**: Integrated call-to-action for appointments.

# Wadul Warga Brebes
A community reporting platform for the Brebes region.

## Tech Stack
* **Frontend**: HTML, CSS (Montserrat Typography)
* **Backend**: Python/Node.js
* **Database**: MySQL/PostgreSQL

## Key Features
* Real-time citizen reports.
* Anonymous submission for privacy.
* Dashboard for local authorities.
